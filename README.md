# CrSDK_MCP

## Usage

1. Get Document ZIP of CrSDK
2. Unzip it
3. Copy html directory to same directory of helpguid_mcp.py ( = This directory )
4. ```$ pip install -r requirements.txt```
5. Generate DB by ```$ python build_help_guilde.py```
6. Run VSCode and open this directory (Ctrl+K Ctrl+O)
7. Execute this MCP
8. Asy anything about CrSDK's API to AI
